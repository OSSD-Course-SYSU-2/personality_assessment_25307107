# 💕 恋爱性格测试

一款基于鸿蒙HarmonyOS开发的恋爱性格测试应用，通过科学的MBTI性格测试，帮助用户了解自己在恋爱中的真实模样，找到最适合的另一半，收获甜蜜爱情。

## ✨ 功能特点

### 🔍 恋爱性格测试
- 16道精心设计的恋爱场景题目
- 基于MBTI四维度（E-I、S-N、T-F、J-P）科学分析
- 测试进度实时显示，支持返回修改答案

### 💘 恋爱性格分析
- 16种恋爱性格类型详细解读
- 恋爱中的优势与劣势分析
- 性格特点深度剖析

### 💑 最佳伴侣匹配
- 科学匹配最适合你的恋爱对象类型
- 需要注意的磨合类型提醒
- 理想伴侣特质描述

### 🌹 约会实战建议
- 量身定制的约会技巧
- 恋爱沟通方式分析
- 爱的语言解读
- 恋爱中的挑战与应对

## 📱 页面说明

| 页面 | 说明 |
|------|------|
| Index | 首页，应用入口，展示功能亮点 |
| TestPage | 测试页面，16道恋爱场景题目 |
| ResultPage | 结果页面，展示恋爱性格和最佳伴侣预览 |
| LoveAdvicePage | 恋爱建议页面，完整的恋爱指导 |

## 🗂️ 项目结构

```
entry/src/main/ets/
├── model/
│   └── MBTIModel.ets          # 数据模型定义
├── data/
│   ├── MBTIQuestions.ets      # 恋爱测试题目数据
│   ├── MBTIResults.ets        # 16种性格类型结果数据
│   └── LoveAdviceData.ets     # 16种类型恋爱建议数据
└── pages/
    ├── Index.ets              # 首页
    ├── TestPage.ets           # 测试页面
    ├── ResultPage.ets         # 结果展示页面
    └── LoveAdvicePage.ets     # 恋爱建议页面
```

## 🚀 使用流程

1. 打开应用，进入首页
2. 点击"开始测试，发现恋爱密码"
3. 完成16道恋爱场景题目
4. 查看恋爱性格结果和最佳伴侣预览
5. 点击"查看完整恋爱建议"获取详细恋爱指导

## 🛠️ 开发环境

- **框架**: HarmonyOS ArkTS
- **SDK版本**: 6.0.2(22)
- **目标设备**: Phone
- **运行系统**: HarmonyOS

## 📋 运行说明

1. 使用DevEco Studio打开项目
2. 在 `Tools > Device Manager` 中创建并启动手机模拟器
3. 点击运行按钮，选择目标模拟器
4. 应用启动后自动进入首页
